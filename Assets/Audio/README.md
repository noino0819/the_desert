# The SSand — 오디오 에셋 가이드

## BGM 파일명 규칙
| 파일명 | 용도 | 분위기 |
|--------|------|--------|
| BGM_Prologue | 프롤로그 | 따뜻한 어쿠스틱 기타 |
| BGM_Ch1_Desert | Ch.1 사막 | 밝고 희망적, 휘파람 |
| BGM_Ch1_Oasis | Ch.1 오아시스 | 활기찬 마을 |
| BGM_Ch1_Boss | Ch.1 보스 (사막하마) | 중간 템포, 긴박 |
| BGM_Ch2_Desert | Ch.2 사막 | 뜨겁고 건조한 |
| BGM_Ch2_Oasis | Ch.2 오아시스 | 아기자기, 장난스러운 |
| BGM_Ch2_Boss | Ch.2 보스 (독수리) | 빠른 추격, 긴장감 |
| BGM_Ch3_Desert | Ch.3 사막 | 황혼, 쓸쓸한 |
| BGM_Ch3_Oasis | Ch.3 오아시스 | 서정적, 분열된 느낌 |
| BGM_Ch3_Boss | Ch.3 보스 (늑대) | 3/4 왈츠, 우아+긴박 |
| BGM_Ch4_Desert | Ch.4 사막 | 밤, 단선율 피아노 |
| BGM_Ch4_Oasis | Ch.4 오아시스 | 앰비언트, 연구실 |
| BGM_Ch4_Boss | Ch.4 보스 (과일거북) | 전략적, 카드게임 |
| BGM_Ending | 엔딩 공통 | 조용한 피아노 |

### 2회차(NG) BGM
모든 BGM의 마이너 버전 — 파일명 뒤에 `_minor` 추가
예: `BGM_Ch1_Desert_minor`

## 형식
- BGM: `.ogg` 또는 `.mp3` (루프 가능)
- SFX: `.wav` 또는 `.ogg`
- 샘플레이트: 44100Hz
- 채널: 스테레오(BGM) / 모노(SFX)
